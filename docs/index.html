Convenções

Prefixos de colunas:

  • id: chave primária
  • cd: qualquer tipo de código que represente um dado que não pertence a nenhuma lookup table.
  • nm: nomes (string)
  • pw: senha (password)
  • ds: qualquer string que não seja um nome.
  • bl: valores booleanos, falso ou verdadeiro.
  • dt: datas
  • nr: valores numéricos
